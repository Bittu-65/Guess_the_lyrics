# Lyrics Guessing Game

#### Video Demo: https://youtu.be/kXOHljfd1kQ

#### Description:
Lyrics Guessing Game is a Python-based terminal game that
tests your knowledge of song lyrics using the Genius API.
The player enters a song name and artist, and the program
fetches the real lyrics of that song. The game then picks
random lines from the lyrics and blanks out a random word
for the player to guess.

The game has 5 rounds per session. In each round the player
gets 3 attempts to guess the missing word. If the player is
stuck they can ask for hints. There are 3 levels of hints:
the first hint reveals the first letter of the word, the
second hint reveals the length of the word, and the third
hint reveals both the first and last letter of the word.

The scoring system rewards players for using fewer hints.
Guessing correctly with no hints gives 3 points, with one
hint gives 2 points, with two hints gives 1 point, and
using all three hints gives 0 points even if correct.
